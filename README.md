# TGUI-DarkTheme
TGUI "Texus' Graphical User Interface" ([TGUI](https://tgui.eu/)) theme meant to look like the default theme in dark mode

Example code built with: 
  - Compiler: MSYS2
  - Version: g++.exe (Rev3, Built by MSYS2 project) 12.1.0

SFML:
  - Version: 2.6.1

TGUI:
  - Version: 1.4.1
